Ce projet a pour objectif d’analyser la relation entre les **importations (M)**, le **PIB**, l’**indice des prix à la consommation (IPC)** et une **variable de tendance** sur la période **1960–2017**
⚙️ Méthodologie
1. Collecte des données (1960–2017).  
2. Transformation des variables (logarithmes : log(M), log(PIB), log(IPC)).  
3. Estimation du modèle sous **EViews** et **R**.  
4. Tests économétriques :  
   - Multicolinéarité  
   - Hétéroscédasticité  
   - Autocorrélation (Durbin-Watson)  
   - Normalité des résidus  
5. Interprétation économique des résultats.
6. - **EViews** (analyse statistique et tests de diagnostic)  
- **R Studio** (régression, visualisations et sauvegarde du modèle)  
- **Excel** (préparation et nettoyage des données)
